# UberEats-Clone
Full-stack UberEats clone built using Angular for the frontend and Node.js (Express) for the backend. Includes restaurant listings, user authentication, food ordering, and real-time order tracking features. Developed collaboratively by a 4-member team (2 frontend, 2 backend).
